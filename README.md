# GemRay
Render raytraced images of faceted gemstones
GemRay is an open source port of GemRay for Windows. GemRay reads .gem or .asc files as created by GemCad (www.gemcad.com).

GemRay is written in C++. It uses the FLTK GUI library, www.fltk.org. This version uses FLTK 1.4.4. It should compile on Mac or linux with minimal changes. The author has tested it on Windows 11 compiled with mingw64 msys64 gcc.
